\begin{defn*}{Absolute Value}

For any $x \in \mathbb{R}$ 


\[
| x | \stackrel{\texttt{D}}{=} \begin{cases} x &, \text{ If } x \ge 0\\ -x &, \text{ If } x < 0 \end{cases}
\]

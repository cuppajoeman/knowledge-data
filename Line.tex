\begin{defn*}{Line}
  A line is an object with no curve or thickness and extends indefintely in two directions. In terms of cartesian coordinates any line can be reresented by the following set for some choice of $a, b, c \in \mathbb{R}$:
  \[
  \left\{ \left( x, y \right): ax  +  by   = c \right\}
  \]
\end{defn*}

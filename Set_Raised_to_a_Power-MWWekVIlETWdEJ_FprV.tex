\begin{defn*}{Set Raised to a Power}
For a given set $X$ we may write $X^{n}$ to represent the set $\left\{ \left( x_1, x_2, \ldots, x_{n-1}, x_{n} \right): \forall i \in \left[ n \right], x_{i} \in X \right\}$, the notation is suggestive of 
\[
	\underbrace{X \times X \times \ldots \times X}_{n\text{ times }}
\] 
even though elements from this set would have many nested inner tuples, which do not need to be considered.
\end{defn*}

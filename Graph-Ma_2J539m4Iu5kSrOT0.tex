\begin{defn*}{Graph}
A graph is a pair $G = (V, E)$ where $V$ is a set whose elements are called vertices and $E \subseteq V \times V$ a set whose elements are called edges.

When $(x,y) \in E$ we will write $ x \mathrel{\bullet  \! {-} \! \bullet} y$

\end{defn*}

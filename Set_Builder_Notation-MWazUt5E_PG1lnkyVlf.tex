\begin{defn*}{Set Builder Notation}
Given a predicate $P$, and a set $S$ the set of all elements in $S$ that such that $P$ holds,  on it is written as: 
\[
	\{ x \in S: P(x) = \mathtt{T} \}
\]
\end{defn*}

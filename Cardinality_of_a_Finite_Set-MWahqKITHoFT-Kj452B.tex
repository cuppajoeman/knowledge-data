\begin{defn*}{Cardinality of a Finite Set}
  Given a nonempty set $X$ we say that $X$ is finite if there is some $n \in \mathbb{N} ^{\ge 1}$ such that we have a bijection 
  \[
  f : X\to  \left[ n \right]
  \]
  And we say that $X$ has cardinality $n$ and write $ \left| X \right| = n$. We define $ \left| \varnothing \right| = 0$  
\end{defn*}
